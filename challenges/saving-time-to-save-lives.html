Saving Time to Save Lives

#design #SOP #procedures #DataRepresentation #language #RFP

The Problem

Video here if applicable

For emergency situations in space, every second is crucial.
Currently, the Emergency Operations Procedures for Space are in an instruction manual format that is difficult to decipher.

The Mission

Create an interface to guide astronauts through standard protocols used in response to emergency situations.

Mission Breakdown

Possible approaches

  • Checklist app with visual guides on hand for different procedures
  • A browser-compatible view of the current PDF with rich-media (diagrams, videos, figures) annotations
  • A framework that allows input of annotations

Available Materials

  • Links to sample SOPs
  • Glossaries, references
  • Data models, data structures
